Projekt-Website zu Drupal8 - Metropolen

Anleitung zum Nachbauen

  • Seite anlegen => Impressum
  • Modul easy-breadcrumb für current page titel installieren
  • Taxonomy "Metropolen" anlgen Alias: \kontinent\land
  • Menü aus Taxonomy erzeugen a) Modul "Taxonomy Menü" installieren b) Menü "Metropolen" anlegen c) Taxonomy-Menü > Neu mit Menü "Metropolen" verbinden d) Blocklayout: Menu: Sidebar first - Menüebenen 1-2
  • Inhaltstyp Artikel anpassen -> umbenennen in "Städte" a) Feld Taxonomy - Reference "Metropolen" ergänzen => unbegrenzte Anzahl an Tags b) Input/Output - Ansicht anpassen c) Bearbeiten > Menüeinstellungen - Verfügbare Menüs: Metropolen Menü aktivieren d) Anzeigeeinstellungen > Autor&Date deaktivieren e) Feld image > Dateiodner [bilder]
  • Artikel => Städte erfassen und Menüpunkt erzeugen Achtung: Immer auch übergeordnete Kategorie (Kontinent) anklicken!
  • Modul Linkit zum Verlinken von Artikeln (interne Links) a) Linkit Profil anlegen: all_users b) Linkit Plugin bei Textformat (/admin/config/formats) aktivieren und Icon in CKeditor einfgüen => Artikel Neapel mit Rom und Madrid verlinken
  • Adaptive Themes (Theme Generator) benötigt AT-Tools Modul Design: neuer Karteireiter: AT Theme Generator Doku: http://docs.adaptivethmes.com Struktur > Blocklayout > Site branding > Konfigurieren: SiteLogo, Name der Website, Slogan der Website Themes > metropolen > Einstellungen > Basic Settings: Pfad zum eigenen Logo: Logo-weltkarte.jpg custom.css: .pr-header {background-color: #0B2148; color: #D39870;} Favicon.ico hochladen
  • Blocklayout: user menu ausblenden Sidebar Second: Tools & user Login ausblenden Sidebar First: Metropolen Menü einblenden
  • Aufteilung Header 6:6 ändern in 10:2 Theme > metropolen > Einstellung > Layouts (unten) page (Default) > wide all and min-width 60em tablet: 8:4
  • Bilder position: Theme > metropolen > Einstellungen > Image Settings > Article > Ausrichtung
  • Farbschema: Tipping Point
  • Datei-Browser: Modul: file browser => zur Integration in den Editor => Icon root > libraries > dropzone root > libraries > imageload root > libraries > masonry
  • Paragraphs => zur individuellen Gestaltung von Artikeln a) Paragraphtypes anlegen: bild, bild+text, text+bild, text, paragraph (wichtig!) b) Inhaltstyp "Paragraph" anlegen > Feld: paragraph c) Wichtig: permissions view:xyz aktivieren
